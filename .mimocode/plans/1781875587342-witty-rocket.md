# 方案 B：筛选栏优化 + 四象限优先级筛选

## 目标

将 todo 页的杂糅下拉筛选拆成三行独立筛选，新增 P1-P4 优先级筛选。

---

## 筛选 UI（三行独立 + 可组合）

```
┌──────────────────────────────────────┐
│ [全部]  [已完成]  [待完成]            │  ← 完成状态（互斥单选）
│ [全部]  [P1 紧急] [P2 重要] [P3 紧急] [P4 可做]  │  ← 优先级（互斥单选）
│ [标签筛选 ▼]                          │  ← 下拉（保持现有实现，精简）
└──────────────────────────────────────┘
```

- 使用 TDesign `t-check-tag` 组件，`variant="light"`，选中态自动高亮
- 每行互斥（同一行内只能选一个）
- 行间独立组合：如 P1 + 待完成 同时生效
- 标签行默认收起，点击展开，不占主空间

---

## 数据模型（todo.js data 新增）

```javascript
currentStatusFilter: 'all',     // 'all' | 'completed' | 'uncompleted'
currentPriorityFilter: 'all',   // 'all' | 'p1' | 'p2' | 'p3' | 'p4'
```

原有 `currentTagFilter` / `currentTagName` / `showTagDropdown` 保留不变。

---

## 筛选逻辑（todo.js）

### 新增 `applyFilters()` 合并三路筛选

```javascript
applyFilters() {
  const { allTodos, currentStatusFilter, currentPriorityFilter, currentTagFilter } = this.data;
  let filtered = allTodos;

  // 1. 完成状态筛选
  if (currentStatusFilter === 'completed') {
    filtered = filtered.filter(t => t.completed);
  } else if (currentStatusFilter === 'uncompleted') {
    filtered = filtered.filter(t => !t.completed);
  }

  // 2. 优先级筛选
  if (currentPriorityFilter !== 'all') {
    filtered = filtered.filter(t => (t.priority || 'p2') === currentPriorityFilter);
  }

  // 3. 标签筛选（原有逻辑）
  if (currentTagFilter === 'completed') {
    filtered = filtered.filter(t => t.completed);
  } else if (currentTagFilter === 'uncompleted') {
    filtered = filtered.filter(t => !t.completed);
  } else if (currentTagFilter === 'none') {
    filtered = filtered.filter(t => !t.tags || t.tags.length === 0);
  } else if (currentTagFilter !== 'all') {
    filtered = filtered.filter(t => t.tags?.some(tg => String(tg) === String(currentTagFilter)));
  }

  this.setData({ todos: filtered });
}
```

### 新增事件处理器

```javascript
onStatusFilterChange(e, filterValue) {
  this.setData({ currentStatusFilter: filterValue }, () => this.applyFilters());
}

onPriorityFilterChange(e, filterValue) {
  this.setData({ currentPriorityFilter: filterValue }, () => this.applyFilters());
}
```

### 修改 `selectTagFilter`

标签筛选时调用 `applyFilters()` 替代 `applyTagFilter()`（或合并后移除 `applyTagFilter`）。

## 待办列表项显示优化

在列表项内添加优先级文字标签（当前仅左侧色条，不可见 P2）：

```xml
<view wx:if="{{todo.priority && todo.priority !== 'p2'}}" class="priority-badge priority-{{todo.priority}}">
  {{todo.priority?.toUpperCase()}}
</view>
```

用 CSS 加小圆点 + 文字：

```css
.priority-badge {
  display: inline-flex; align-items: center; gap: 4rpx;
  padding: 2rpx 12rpx; border-radius: 12rpx;
  font-size: 22rpx; font-weight: 500;
}
.priority-badge.priority-p1 { background: #fef0ef; color: #e34d59; }
.priority-badge.priority-p3 { background: #fff3e0; color: #ff9800; }
.priority-badge.priority-p4 { background: #f5f5f5; color: #999; }
```

（P2 默认不显示徽章，因为它是默认值）

## 验证

1. 编译运行小程序，进入 todo 页
2. 默认显示全部待办，筛选行全部选项高亮
3. 点「已完成」→ 只显示已完成
4. 点「P1」→ 只显示 P1
5. 点「P1」+「待完成」→ 只显示未完成的 P1
6. 点「标签」下拉选标签 → 多维度筛选叠加
7. 切换 Tab 回来 → 筛选状态重置为默认
