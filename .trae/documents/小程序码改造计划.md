## 接口设计

```
GET /collab/qrcode?shareCode=ABC123&auto=0&token=xxx

参数：
- shareCode: 6位邀请码
- auto: 0 或 1，是否直接加入
- token: 用户认证token（可选，用于验证权限）

返回：直接返回图片二进制流（PNG格式）
```

---

## scene 参数格式

小程序码的 scene 参数支持两种格式：

### 格式1：键值对格式（推荐）

```
code=ABC123&auto=0   （需要审批）
code=ABC123&auto=1   （直接加入）
```

### 格式2：简洁格式

```
ABC123    （需要审批）
ABC123_1  （直接加入）
```

前端 `app.js` 和 `join-collab.js` 中已支持解析这两种格式。
